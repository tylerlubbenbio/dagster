2. Add any extra references needed in PROJECT_INFO.
3. Delete any things not needed in the STARTHERE
4. Go to the Cloud folder and delete either the Linux or the Windows hooks and then rename it Hooks.
5. Go to the settings.json and if it's Linux for Windows, cut it and put it into the cloud directory then delete the folder
